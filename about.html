About Code Kid Code

At Code Kid Code, we believe that every child can learn to code. Our mission is to make programming fun and accessible for kids aged 7-9 using Microsoft MakeCode and Minecraft Education.

Our courses focus on teaching basic programming concepts such us variables, loops, conditions, and problem-solving while encouraging creativity and teamwork.
